‚öôÔ∏è 6. IMPLEMENTATION INSTRUCTIONS

Step 1: Add CSS Variables

Create or update your `globals.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

:root {
  --background: #fafafa;
  --foreground: #1a1a1a;
  --card: #ffffff;
  --primary: #0055FF;
  --muted-foreground: #86868b;
  --border: rgba(0, 0, 0, 0.08);
  --destructive: #ff3b30;
  --success: #34c759;
}

.dark {
  --background: #0a0a0a;
  --foreground: #fafafa;
  --card: #141414;
  --primary: #5B8EFF;
  --muted-foreground: #a1a1aa;
  --border: #27272a;
}

@layer base {
  body {
    @apply bg-background text-foreground;
  }
}
```

Step 2: Configure Tailwind

Update `tailwind.config.js`:

```js
module.exports = {
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        background: 'var(--background)',
        foreground: 'var(--foreground)',
        card: 'var(--card)',
        primary: 'var(--primary)',
        border: 'var(--border)',
      }
    }
  }
}
```

Step 3: Use in Components

```tsx
// Light and dark mode automatically work
<div className="bg-background text-foreground">
  <div className="bg-card border border-border">
    <span className="text-primary">Blue text</span>
  </div>
</div>

// Status colors
<div className="bg-green-50/30 dark:bg-green-950/20">
  <span className="text-green-600 dark:text-green-400">Optimal</span>
</div>
```
